# foxtrotjellyfish.github.io

Public splash page for [Foxtrot Jellyfish](https://github.com/foxtrotjellyfish) — open-source agentic systems engineering.

## What's here

Static `index.html` served via GitHub Pages. No build step, no framework, no dependencies.

## Projects

- **[The Substrate](https://github.com/foxtrotjellyfish/the-substrate)** — Eight protocols for AI-augmented knowledge workspaces. Pure markdown specification.
- **[The Cortex](https://github.com/foxtrotjellyfish/the-cortex)** — Self-hosted Elixir/OTP orchestration engine. Reference implementation of the Substrate.

## Deployment

Push to `main` on this repo. GitHub Pages serves from root.
